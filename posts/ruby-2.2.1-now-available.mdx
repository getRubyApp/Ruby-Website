Ruby 2.2.1 is now available for iOS, macOS, and visionOS - with key fixes and improvements. It also brings all of the features of Ruby 2.2 to macOS and visionOS for the first time, which you can read more about [here](https://rubyapp.co/updates/ruby-2.2-now-available).

# Changelog

- Reintroduced support for iOS 17
- Fixed dark mode color issue with in-app browser
- Resolved some visual bugs
- Updated trending page behavior

Ruby is available on the App Store for $3.99.
